# Картацион.рф — интерактивная карта для Twitch (донаты → метры маршрута)

**Картацион.рф** — бесплатная интерактивная карта для стримеров: зрители отправляют донаты, а вы проходите маршрут на карте в реальном времени по правилам челленджа.

Демо/сайт: https://xn--80aayfsflq1b.xn--p1ai  
(Картацион.рф — официальный адрес проекта)


## Что такое Картацион
Картацион делает донаты “действием”: сумма доната превращается в расстояние (метры), а стример двигается по карте и старается пройти максимум за ограниченное время.
Форматы:
- Карта‑челлендж на время.
- Донат‑аукцион маршрута (чат направляет, куда идти дальше).

## Основные возможности
- Интерактивная карта + построение маршрута  сраззу на карте картациона, при помощи open-source инструментов, доступно по всему миру.  .
- “Цена за метр” → автоматический пересчёт доната в метры.
- Управление движением по направлениям (N / S / E / W).
- Таймер челленджа (обратный отсчёт).
- Лог событий в реальном времени (ходы, таймер, броски).
- Кубик рандомайзер (рандом для доната/хода), так же имеется четное-нечетное. Еще есть страница (https://xn--80aayfsflq1b.xn--p1ai/test-random.html) что бы пользователи могли посмотреть как работает наш рандомайзер. 
- Undo (шаг назад), если ошибся ходом.
- QR‑код маршрута, чтобы открыть итоговый путь в навигаторе/картах, подходит как для Android смартфонов так и для iOS. При сканировании QR открывается приложение Яндекс.Карты.

## Как использовать на стриме
1. Открой Картацион.рф в браузере.
2. Настрой цену за метр и таймер.
3. Запусти челлендж.
4. Зрители донатят → маршрут растёт → ты двигаешься по карте.
5. В конце — QR‑код маршрута (можно показать зрителям и пройти “в реале”).

## Примеры сценариев для стримов
- “Дойду до точки X за 60 минут: чат решает путь донатами”.
- “Каждые N рублей — один шаг + шанс кубика”.
- “Маршрут‑аукцион: зрители выбирают направление донатами”.

## FAQ
**Это бесплатно?**  
Да, Картацион.рф задуман как бесплатная альтернатива для стримеров.

**Нужны ли плагины/OBS?**  
Можно использовать как отдельную вкладку в браузере или как окно/источник в OBS (по желанию).

**Можно ли откатить ход?**  
Да, есть Undo (шаг назад).
